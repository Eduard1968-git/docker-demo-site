Over dit project

Dit is de demo-site die gebruikt wordt voor de Docker-opdracht van Digitale Werkomgeving 2.

De site wordt geserveerd via een Nginx-container op poort 8888.

Docker informatie

  • Base image: nginx:1.24.0-alpine
  • Poort: 8888 → 80
  • Buildtijd: buildtime.txt
